## 🏥CareMe의 한줄 소개

UN의 지속가능발전목표(SDGs) 중 3번째 목표인 "건강과 웰빙" 달성과 현대인의 건강염려증 해소를 위해 시작된 CareMe는 체계적인 건강관리와 심리적 안정을 동시에 지원하는 플랫폼입니다.

## ✨CareMe 제작 배경

<b>건강 염려증</b>은 자신에게 심각한 질병이 있다는 생각과 불안감을 갖고 집착하는 상태입니다. 사소한 신체증상을 지나치게 비관적으로 받아들이고 의사의 진단도 믿지 않으며, 불안감 때문에 이 병원 저 병원 돌아다니며 각종 검사를 받습니다. 저희의 서비스는 이러한 건강 염려증 문제를 해결하기 위해 테스트를 통해 자신의 상태를 확인하고 필요한 조치를 취하여 더 건강한 삶을 살아가도록 방향지침을 제공합니다.

## 🎯 해결하고자 하는 문제

![image](https://github.com/user-attachments/assets/3b6488f4-c53e-4cdf-aaab-e0ea197ea414)


UN의 지속가능발전목표(SDGs) 중 3번째 목표인 "건강과 웰빙" 달성과 현대인의 건강염려증 해소를 위해 시작된 CareMe는 체계적인 건강관리와 심리적 안정을 동시에 지원하는 플랫폼입니다.

**건강염려증**

> 자신에게 심각한 질병이 있다는 생각과 불안감을 갖고, 집착하는 상태
> 

> 사소한 신체증상을 지나치게 비관적으로 받아들이고 의사의 진단도 믿지 않으며, 불안감 떄문에 이 병원 저 병원 돌아다니며 각종 검사를 받는 행위를 반복하게 됨
> 

1. 건강염려증의 증가
    - 코로나19 이후 급증한 건강염려증
    - 과도한 의료정보 검색으로 인한 사이버콘드리아
    - 불필요한 병원 방문과 의료비 지출
    - 지속적인 건강 불안으로 인한 삶의 질 저하
2. 체계적인 건강관리의 어려움
    - 산재된 건강 정보 관리
    - 비일관적인 생활습관 기록
    - 예방적 건강관리의 부재

## 💡 솔루션

CareMe는 다음과 같은 기능을 통해 건강염려증 해소와 지속가능한 건강관리를 지원합니다.

1. 건강염려증 관리
    - 건강 예민도 측정 및 추적
    - 증상 기록을 통한 객관적 건강 상태 파악
    - 과거 증상과 현재 상태 비교 분석
    - 반복되는 건강 불안 패턴 인식 지원
    - AI 기반 건강 상담을 통한 불안 해소
2. 체계적인 건강 기록
    - 일상 증상 추적
    - 생활습관 모니터링
    - 의료 기록 통합 관리
3. 예방적 건강관리
    - 건강검진 일정 관리
    - 예방접종 현황 확인
    - 영양제 복용 관리

## 💪 차별성

1. 건강염려증에 특화된 기능
    - 증상 기록과 회고를 통한 불안 패턴 인식
    - 과거 증상 분석을 통한 객관적 건강 상태 파악
    - AI 상담을 통한 즉각적인 불안 해소
2. 통합적 건강관리
    - 신체적 건강과 정신적 웰빙의 통합 관리
    - 예방적 건강관리와 건강 불안 해소의 결합
    - 개인 맞춤형 건강관리 솔루션 제공
  
## 💊SDGs 3번 목표(Good-Health and Well-Being) 달성 방안

1. 보편적 의료보장 달성
- 기능: 건강기록 통합관리, 병원 진료기록 분석
- 목표: 의료서비스 접근성 향상, 불필요한 중복진료 방지
2. 정신건강 증진
- 기능: 건강 예민도 측정, AI 상담, 불안 패턴 분석
- 목표: 건강불안 감소, 정신건강 증진
3. 비전염성 질환 예방
- 기능: 생활습관 모니터링, 맞춤형 건강관리 플랜
- 목표: 질병 조기 발견, 만성질환 예방
4. 예방의료 강화
- 기능: 건강검진/예방접종 알림, 정기검진 관리
- 목표: 질병 예방, 조기발견율 향상
5. 의료정보 접근성 개선
- 기능: 개인 건강데이터 분석, 맞춤형 건강정보 제공
- 목표: 의료정보 불평등 해소, 건강관리 역량 강화
